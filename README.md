# ðŸ“¸ BW Converter (Aplikasi Foto Hitam Putih)

**BW Converter** adalah aplikasi desktop sederhana berbasis Python yang berfungsi untuk mengubah gambar berwarna menjadi hitam putih (grayscale). Aplikasi ini mendukung pemrosesan banyak gambar sekaligus (*batch processing*) melalui antarmuka grafis (GUI) yang mudah digunakan.

## âœ¨ Fitur Utama
* **Antarmuka Sederhana (GUI):** Tidak perlu mengetik kode, cukup klik tombol.
* **Batch Processing:** Bisa memproses banyak foto sekaligus dalam satu klik.
* **Format Didukung:** JPG, JPEG, PNG, BMP, WEBP.
* **Organisir Otomatis:** Hasil foto disimpan di folder terpisah (`Hasil_BW`) agar tidak menimpa file asli.
* **Portable:** Bisa dijadikan file `.exe` dan dijalankan di Windows tanpa instal Python.
